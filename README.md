## Pré-requisitos  
- Python 3.x (recomenda-se Python 3.7 ou superior)  
- Pygame (versão compatível com seu sistema)  

## Instalação  
1. Clone este repositório:  
   ```bash
   git clone https://github.com/DevMarlon-ai/Python-Jogos-Aula.git
   cd Python-Jogos-Aula
```


2. Instale o Pygame:

   ```bash
   pip install pygame
   ```

## Execução dos jogos

### Para executar **Acerte o Alvo**

No diretório principal do repositório, execute:

```bash
python acerte-alvo.py
```

Siga as instruções na tela para jogar.

### Para executar **Pong**

No diretório principal do repositório, execute:

```bash
python pong.py
```

Use as teclas conforme informado (por exemplo, seta para cima/baixo ou W/S) para jogar.
